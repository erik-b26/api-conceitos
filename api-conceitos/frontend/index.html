Bem-vindo ao Explorador de Conceitos de Testes de Software

Este frontend permite explorar conceitos relacionados a testes de software armazenados em um backend simples.

Use as páginas para navegar:

  • Home: Esta página, explicando o propósito.
  • Geral: Lista todos os conceitos disponíveis.
  • Pesquisa: Busque por termos específicos nos conceitos.
  • Random: Descubra um conceito aleatório.

O backend roda em http://localhost:3000 e serve dados de testes de software.
